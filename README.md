- # Student Score Prediction

## Description
This project predicts student marks based on hours studied and attendance using Linear Regression.

## How to Run
1. Install required packages:
   pip install -r requirements.txt
2. Run the model training:
   python train_model.py
3. Run Flask server:
   python app.py
4. Test API:
   - Use Postman or browser
5. Optional: Open index.html for frontend

## Input
- hours_studied: integer
- attendance: integer (%)

## Output
- Predicted Marks
